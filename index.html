About Utils

Utils is a dedicated CDN for web components, providing modular and reusable components for modern web applications.

Features

  • ✅ Fast and reliable CDN
  • ✅ Plug-and-play web components
  • ✅ Open-source and customizable
  • ✅ Seamless integration with any framework

How to Use

Simply include the following script in your HTML:

<script src="https://cdn.utils.com/component.js" defer></script>
